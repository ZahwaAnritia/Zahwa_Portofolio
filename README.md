# Zahwa Anritia - Personal Portfolio 🚀

Selamat datang di repository portofolio saya. Website ini dibangun menggunakan **React.js**, **TypeScript**, dan **SCSS** untuk menampilkan proyek teknis serta perjalanan karier saya sebagai **Full-Stack Web Developer**.

🔗 **Live Demo:** [https://zahwa-anritia.vercel.app](https://zahwa-anritia.vercel.app) *(Ganti dengan link Vercel kamu nanti)*

## 🛠️ Tech Stack Website Ini
- **Frontend:** React.js, Material UI (MUI)
- **Styling:** SCSS / SASS
- **Icons:** FontAwesome & Material Icons
- **Deployment:** Vercel

## 📂 Featured Projects
Saya berfokus pada pengembangan aplikasi Web dan Mobile dengan berbagai tech stack:
1. **Smart Attendance & IoT Dashboard** - Node.js, Express, & Socket.io
2. **KTP Management System** - Java Spring Boot & JQuery Ajax
3. **Sakuku: Finance Management** - Android Native (Kotlin)
4. **CineData Platform** - Node.js Native API (Fundamental)

## 🚀 Quick Start
Untuk menjalankan website ini di lingkungan lokal:
1. Clone repository ini.
2. Jalankan perintah berikut:
    ```bash
    npm install
    npm start
    ```
